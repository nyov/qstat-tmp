QStat Derived Works

Several clever people have used QStat to make their own programs. I have only tried a few of the programs listed below.

If you distribute a program or script that uses QStat, please send me email and I'll list your package on this page.


UgLy Game
Search Engine 		UgLy Game Search Engine quickly searches the local network for game servers or checks your list of favorite servers. HTML status pages are generated for each server and a summary page lists all servers with graphs of usage patterns. It will also generate a QStat-style server list for use with other programs such as ServerQuery. This is a small Perl script you can run in the background of a Linux or Unix system.

Frag Meter A player ranking system using a select set of servers. A tournament manager is in the works by the same author.

Quake status IRC bot IRC bot on irc.enterthegame.com #clanpf
It's called Vbot. To get info you just go to that channel and type "VV qstat q3s ip:port P" The P at the end is optional for getting a player list. The link is to the author's server status page.

phpQStat A PHP class for querying a game server for status, rules, and player info.

Game Trakker Tracks server usage for your game servers. Generates current status pages and historical usage graphs. Uses Perl, QStat, and MRTG. GPLd.

Gamers Home Clans Set up a server status page for your Clan. They host the page and do all the work. Open sourced.

QStat4Eggdrop A script to use QStat with IRC bots. Get Quake server status redirected to an IRC channel. Supports QuakeWorld.

BloodyVista Web based server browser and launcher for QuakeWorld. Will work on any Unix with a QuakeWorld client (ie. Linux). Pretty neat system, uses a Perl CGI script and QStat to perform the queries. Offline - The author's web page is off-line and I can't find the source any where. The link above is someone running Bloody Vista. Here's another location.

TalitaSpy Windows server browser and game launcher for Quake 2. Author used QStat source as reference.

Server Watch Win 95/NT system tray utility. Pings one server and displays the current number of players.

QstatList Automates and simplifies creation of web pages that list Quake servers. Version 3.7 available, takes advantage of the new features in QStat 2.1. Offline - QStatList home page appears to be off-line. The link will take you to the Gameaholic page for the 3.4 version.
Quickspy2 A Quake2 server browser and launcher for Linux console.

QSMon Perl CGI script that queries Quake servers and formats the results into an HTML status page. Quake 1, 2, 3, and Half-Life. New URL!

Qcheck Displays information on a Quake server and the players on it in a format similar to the Unix program Top. For Linux and Windows 95.

LQWSPY Linux QuakeWorld server browser and launcher.

XQF Linux server browser and launcher.

Qplug for Linux Launch Quake directly from a web server list. Uses portions of QStat code. Version 0.4 is available.
