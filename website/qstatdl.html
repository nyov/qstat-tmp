The Unix download contains source only. The Windows download includes a pre-compiled Windows binary in addition to the source code. Otherwise the files are the same in the two archives except that the text files in the Windows version use <cr><lf> to terminate lines, and HTML files in the Windows version have a ".htm" file extension.
