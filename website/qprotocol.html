Quake Protocol Information

Some places containing information about the Quake network protocol. If you know of any others, please let me know.

  • Quake2 Net Specs
    Work in progress on the Quake II network protocol. Currently does not include info on the server status packets.
  • Unofficial Quake Specs
    No QuakeWorld information.
  • Quake World Network Protocol Specs
    This is partially obsolete since it documents the pre-1.5 QuakeWorld (when you had to login to a master server.)
  • QuakeWorld Master Protocol
    Documents the pre-1.5 QuakeWorld master server protocol.
  • Unofficial Quake Network Protocol Specs
    Only applies to versions 0.91, 0.92, 1.00 and 1.01 of Quake.
  • Official Unreal server query
    Straight from the programmers themselves.

I don't know of any published information on the QW or Q2 protocol, nor have I written any myself.

Of course, you can always read the QStat code to figure out how things work.
