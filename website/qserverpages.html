QStat Server Pages

There are enumerable pages using QStat to generate game server status. If you acknowledge the use of QStat on your page, Thank You!

Here's a sample of some Quake server pages. They don't all use QStat.

5 Assed Monkey

Status pages for a pair of Tribes and Tribes 2 servers.
Rob's Quake List

Rob uses a CGI script to track status for a number of Quake 3 servers. Rob also runs an IRC bot that shows Quake server info. See the derived works page for info.
The Champions League for Quake

This site is amazing. Tracks players stats across 100,000+ Internet servers. Raw data gathered with QStat.
Gameaholic

Lists for many different game servers.

Gamers Home

 Gamer news site with personalized server status.

Gamers Home Clans

Set up a server status page for your Clan. They host the page and do all the work. Open sourced.
The Polish Quake Server List

Quake is alive and well in Poland. Not many servers, but there are people playing at 3:00 AM
EarthQuake Server Watch

Dutch servers.
QStatList Statistics

Stats for Internet Quake usage!
