QStat Downloads

The Unix download contains source only. The Windows download includes a pre-compiled Windows binary in addition to the source code. Otherwise the files are the same in the two archives except that the text files in the Windows version *should* be using <cr><lf> to terminate lines.

Production Release (Tested and Stable) - November 4, 2006

  • QStat 2.11 for *nix/Source Code (~252Kb) - gzipped tar file
  • QStat 2.11 for Windows Compiled 32bit (~199Kb) - zip file

Development Release (SVN)

SourceForge Project Page - Please note that this version, although supporting newer game protocals, may be unstable.

SVN version currently includes (as of August 19th 2007)
- Bug Fixes
- ETQW support

You can browse the latest source to view change comments. Or If you wish to download and compile the latest SVN trunk, then you may do so anonymously via the following checkout command:

svn checkout https://qstat.svn.sourceforge.net/svnroot/qstat/trunk/qstat2 qstat2
